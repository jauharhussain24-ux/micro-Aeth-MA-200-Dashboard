MA200 Dashboard - Portable Windows build
==========================================

No Python install required on the other laptop.

Quick start
-----------
1. Copy "MA200 Dashboard.exe" to any folder (or unzip the whole package).
2. Double-click "MA200 Dashboard.exe".
3. Your browser opens to http://127.0.0.1:8000/
4. Plug in the MA200, click Refresh, choose the COM port, click Connect.

Keep the black console window open while you use the dashboard.
Close that window to stop the server.

Requirements on the other laptop
--------------------------------
- Windows 10 or 11 (64-bit)
- Chrome, Edge, or Firefox
- USB port for the MA200
- No internet connection needed

If Windows SmartScreen appears
------------------------------
Click "More info", then "Run anyway". This app is not code-signed.

Troubleshooting
---------------
- If the browser says the site cannot be reached, wait a few seconds and refresh.
- If no COM port appears, check the USB cable and click Refresh again.
- Port 8000 must be free. Close any other copy of the dashboard first.
